## Generating it locally

```bash
cd config-go
git submodule update --init
go doc ./...                   # standard godoc in the terminal
go install github.com/princjef/gomarkdoc/cmd/gomarkdoc@latest
gomarkdoc -o docs/api.md ./... # markdown copy
```
